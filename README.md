# 591 看房助手

在 591 售屋／租屋列表頁標記「已看過」與「有興趣」的物件，重複瀏覽時自動灰化已看過的房源，讓你每次只看新內容。

[![Ko-fi](https://img.shields.io/badge/支持開發者-Ko--fi-f39c12?logo=ko-fi&logoColor=white)](https://ko-fi.com/lbeeon)

---

## 功能

- **標記已看過**：點擊卡片上的按鈕，灰化該物件，下次瀏覽自動套用
- **標記有興趣**：加上黃色外框，方便追蹤心儀物件
- **跳到未看過**：工具列顯示未看過筆數，一鍵滾動到第一筆新物件
- **標記本頁全部已看**：一鍵清空本頁新物件
- **清單管理頁**：查看並管理「有興趣」與「已看過」的完整清單，可編輯備註、個別移除
- **Detail 頁面狀態面板**：進入物件頁時也能標記狀態

支援 `sale.591.com.tw`（售屋）與 `rent.591.com.tw`（租屋），資料各自獨立儲存。

---

## 安裝方式

> Chrome Web Store 版本上架中，目前可手動載入。

1. 下載或 clone 本 repo
2. 開啟 Chrome，前往 `chrome://extensions/`
3. 開啟右上角「開發人員模式」
4. 點「載入未封裝項目」，選擇本資料夾
5. 前往 [591 售屋網](https://sale.591.com.tw) 或 [591 租屋網](https://rent.591.com.tw) 即可使用

---

## 資料儲存

所有標記資料儲存於瀏覽器的 `localStorage`，**不會上傳至任何伺服器**，僅限本機使用，不跨裝置同步。
